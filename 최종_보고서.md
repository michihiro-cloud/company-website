# Company Website 초기 구축 보고서

## 1. 작업 개요

- 목적: 회사 홈페이지 초기 기본 디자인 레이아웃과 뼈대 코드 생성
- 접근 방식: 에르메스 기획안을 바탕으로 React + Vite + Tailwind CSS 기반 구조 생성
- 현재 상태: 전체 프로젝트 폴더, 기본 페이지, 컴포넌트, 더미 데이터, 문서 템플릿 생성 완료

## 2. 생성된 기술 스택

- React
- Vite
- Tailwind CSS
- React Router
- PostCSS

## 4. 페이지 및 컴포넌트 설명

- `Home.jsx`: 히어로 섹션, 서비스 카드, 포트폴리오 카드 포함
- `About.jsx`: 회사 소개 섹션 템플릿
- `Services.jsx`: 서비스 목록 페이지
- `Portfolio.jsx`: 포트폴리오 그리드 페이지
- `Contact.jsx`: 문의 정보 + 연락폼
- `Header.jsx` / `Footer.jsx`: 공통 네비게이션 및 푸터
- `Hero.jsx`: 초기 랜딩 히어로 배너
- `ServiceCard.jsx` / `PortfolioCard.jsx`: 재사용 가능한 카드 컴포넌트
- `ContactForm.jsx`: 더미 문의 폼
- `placeholderData.js`: 더미 텍스트 및 이미지 URL 관리

## 5. 더미 데이터 및 콘텐츠

- `placeholderData.js`에서 더미 제목, 부제, CTA, 서비스 리스트, 포트폴리오 샘플, 연락처 정보를 정의함
- `docs/` 폴더에 실제 회사 콘텐츠로 채워 넣을 수 있는 Markdown 템플릿을 생성함

## 6. 다음 작업 제안

1. `package.json` 설치
   - `npm install`
2. 로컬에서 실행 확인
   - `npm run dev`
3. `public/images/` 폴더에 실제 로고 및 히어로 이미지 추가
4. `docs/` Markdown 콘텐츠를 회사 실제 정보로 교체
5. 필요 시 React Router 내비게이션에 모바일 메뉴 추가
6. GitHub 저장소 초기화 및 Vercel 자동 배포 설정

## 7. 참고

- `.clinerules`에 따라 변경 전 사용자 승인 요청을 모두 진행했음
- 현재 작업은 초기 설계 및 뼈대 코드 생성 단계이며, 실제 콘텐츠 교체는 이후 단계에서 진행해야 함
